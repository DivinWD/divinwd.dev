# divinwd.dev

Source code for the **DivinWD** project landing page:

> **DivinWD: Exploring the Diversity of Scientific Publications in Wikidata**  
> Exploring diversity indicators for scientific publications using Wikidata and external bibliographic sources.

This repository is a small static website (HTML + CSS + images) intended to be deployed at [divinwd.dev](https://divinwd.dev/).
